HackUPC's baggage management system.

## Setup

Needs: PHP 5.4.X

- `git clone https://github.com/hackupc/baggage && cd baggage`
- `Create one database and setup .env for database connection`
- `php artisan migrate:refresh`
- `php artisan serve`
